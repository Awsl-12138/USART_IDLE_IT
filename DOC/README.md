本工程用于测试PY32F030K28U6的coremark跑分
可以根据此工程文件进行修改测试其他mcu的coremark得分
推荐运行mcu配置要求：flash>=16KB,ram>=4KB
（注意，程序中设置的py32f030主频为32mhz.默认TX引脚为B6脚）
结果：
PY32F030K28U6 @48MHz Cortex m0
score: 56.265 Iterations/Sec
score per MHz: 1.172
